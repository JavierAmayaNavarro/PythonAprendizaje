Ejercicios que se van a realizar 

1. Crear un procedimiento que reciba un nombre y muestre un saludo.

2. Devolver el cuadrado de un número usando una función.

3. Función para sumar dos números y procedimiento para mostrar el
resultado.

4. Función para verificar si un número es par.

5. Procedimiento que imprima una tabla de multiplicar.

6. Suma de números primos
a. Crear un método que verifique si un número es primo.
b. Otro método que sume todos los números primos entre 1 y N.

7. Número perfecto
a. Escribir un método que determine si un número es perfecto.
b. Usar otro método que imprima todos los números perfectos hasta
un límite dado.

8. Crear un programa que calcule el mcd de un número de forma iterativa y
otro programa de forma recursiva.

9. Crear un programa que calcule fibonacci de forma iterativa y otro
programa de forma recursiva.

10. Conversión de unidades
a. Crear métodos para convertir entre kilómetros ↔ millas, Celsius
↔ Fahrenheit, kilogramos ↔ libras.
—-------------------------------------------------------------------------------------------------------
11. Rellenar vector: Crea un programa que pida 10 números enteros y los
guarde en un vector. Luego, muéstralos en orden inverso.

12. Suma de elementos: Calcula la suma y la media de los elementos de un
vector de 15 posiciones.

13. Mayor y menor: Encuentra el número mayor y el menor dentro de un
vector de 10 números aleatorios.

14. Buscar número: Permite al usuario introducir un número y comprueba si
está en el vector.

15. Contar ocurrencias: Dado un vector de enteros, cuenta cuántas veces
aparece un número introducido por el usuario.

16. Desplazamiento: Desplaza todos los elementos del vector una posición a
la derecha.

17. Ordenar vector: Ordena un vector de 10 números en orden ascendente
sin usar funciones predefinidas.

18. Eliminar duplicados: Crea un vector con números aleatorios y genera
otro sin duplicados.

19. Matriz identidad: Crea y muestra una matriz identidad de 5x5.

20. Suma de matrices: Suma dos matrices de 3x3 introducidas por el
usuario.

21. Transpuesta: Calcula y muestra la transpuesta de una matriz de 4x4.

22. Mayor de cada fila: Para una matriz de 4x4 aleatoria, muestra el mayor
valor de cada fila.

23. Matriz de multiplicación: Crea una matriz de 10x10 que contenga las
tablas de multiplicar del 1 al 10.

24. Diagonal secundaria: Muestra los elementos de la diagonal secundaria
de una matriz cuadrada de 5x5.

25. Lista de enteros: Crea una lista de enteros, añade elementos al principio
y al final, y muéstrala.

26. Eliminar elemento: Elimina un elemento específico de una lista (si
existe).

27. Insertar ordenado: Inserta números en una lista manteniendo el orden
ascendente.

28. Contar elementos: Cuenta cuántos elementos tiene una lista enlazada y
muestra el resultado.

29. Funciones matemáticas: Crea una función que calcule el factorial de un
número y otra que calcule si un número es primo.

30. Menú modular: Crea un programa con un menú que permita:
● Llenar un vector,
● Mostrarlo,
● Ordenarlo,
● Buscar un número.
Cada opción debe implementarse en una función o procedimiento
independiente
